# 장전 주문 플랜 가이드

- 기준일: 2026-04-22
- 명령: `/장전플랜` 또는 `/리포트 장전플랜`
- 한 줄 요약: 오늘 아침 어떤 종목을 몇 주, 얼마에 넣을지 + 보수적으로 갈지 공격적으로 갈지를 자동으로 계산해줍니다.

---

## 이 기능이 하는 일

`/장전플랜`은 단순 후보 나열이 아닙니다. 실행 전 다음 세 가지를 동시에 반영합니다.

1. **적응형 운용 강도 결정** — 최근 14일 실현손익·승률·미체결 패턴을 보고 오늘 보수적으로 갈지, 표준으로 갈지, 기회를 더 열지 자동 판단합니다.
2. **후보 종목 선별 + 투자 계획** — 점수 상위 종목 중 진입 구간·손절·목표가가 유효한 것만 추립니다.
3. **수량·투자금 계산** — 설정한 투자금과 가용 현금, 슬롯 수를 기준으로 주문 수량과 예상 손익까지 계산합니다.

---

## 사전 준비 (처음 한 번만)

장전플랜을 쓰려면 투자금 설정이 반드시 되어 있어야 합니다.

```
/투자금 300만원 3 8 안전형 5
```

| 항목 | 예시 값 | 설명 |
|------|---------|------|
| 투자금 | 300만원 | 운용 시드 |
| 분할 횟수 | 3 | 한 종목에 최대 몇 번 분할 매수 |
| 목표 수익률 | 8 | % (익절 기준) |
| 투자 성향 | 안전형 | 안전형 / 균형형 / 공격형 |
| 일손실 한도 | 5 | % (당일 손실이 이 이상이면 신규 진입 차단) |

## 매일 사용하는 순서

### 1단계 — 시장 먼저 확인 (2분)

```
/경제
/시장
/브리핑
```

> 리스크 신호가 높으면 장전플랜이 자동으로 보수 모드로 전환됩니다. 억지로 넣을 필요 없습니다.

### 2단계 — 장전플랜 실행 (1분)

```
/장전플랜
```

또는 `/리포트` → **장전플랜** 버튼

### 3단계 — 출력 해석

출력 상단에 오늘의 운용 조건이 표시됩니다.

```
투자성향  안전형 · 시장모드  균형
운용강도  보수적 · 최근 손실 누적으로 진입 기준 강화
가용현금  2,850,000원 · 신규주문 가능  1,500,000원
신규 슬롯  2건 · 기준점수  72점 → 적응형  77점
점수 기준일  2026-04-22
최근 14일 실현손익  -45,000원 · 매도 3건 · 승률 33.3%
```

**운용강도 판독표**

| 강도 레이블 | 상황 | 행동 방향 |
|------------|------|----------|
| 보수적 | 최근 손실 누적 또는 승률 저조 | 점수 기준 상향, 주문 수 축소 |
| 표준 | 성과 중립 구간 | 설정 그대로 운용 |
| 기회 포착 | 최근 승률 양호, 손익 흑자 | 진입 기준 소폭 완화, 슬롯 추가 |
| 수익 유지 | 단기 성과 매우 양호 | 승리 종목 비중 유지, 신중 추가 |

> **보수적** 상태에서 억지로 더 넣으면 회복보다 손실 확대 가능성이 높습니다. 플랜이 주문 수를 줄였다면 그대로 따르는 것이 맞습니다.

### 4단계 — 주문 실행

각 후보마다 아래 정보가 나옵니다.

```
1. 삼성전자  005930
- 판단 즉시 · 점수 78.4 · 손익비 2.1:1
- 매수주문  10주 x 71,200원 = 712,000원
- 진입구간  70,500원 ~ 72,000원
- 손절      68,800원 (-3.4%)
- 1차매도   5주 @ 77,000원 (+8.1%) · 기대 +29,000원
- 2차매도   5주 @ 82,000원 (+15.2%) · 기대 +54,000원
- 주문메모  시초가 과열 갭 출발 시 추격 금지 · 유효시야 3~5거래일
- 근거  ...
```

**각 항목 해석**

| 항목 | 의미 |
|------|------|
| 판단 즉시 | 현재가가 이미 진입 구간 안 → 바로 주문 가능 |
| 판단 눌림대기 | 현재가가 진입 구간 위 → 구간까지 내려오면 매수 |
| 진입구간 | 이 구간 안에서 매수해야 계획이 유효 |
| 손절 | 이 가격 이하로 종가 마감 시 전량 매도 |
| 1차매도 | 전체 수량 절반 익절 목표가 |
| 2차매도 | 나머지 절반 익절 목표가 |
| 손익비 | 기대 이익 / 최대 손실 비율. 1.5:1 미만은 자동 제외 |

### 5단계 — 보유 중 관리

주문 후에는 매일 아래 흐름으로 점검합니다.

```
/보유대응    ← 보유 종목 대응 플랜 (익절/손절/유지)
/자동사이클 점검    ← 자동매매 조건 재확인
```

---

## 후보가 0건일 때

```
오늘은 장전 주문으로 바로 넣을 만한 후보가 없습니다.
```

이 메시지가 나오면 무리하지 않아도 됩니다. 원인은 아래 중 하나입니다.

- 시장 모드가 방어 국면 → 자동으로 기준 점수 상향
- 가용 현금 부족 → `/투자금`으로 증액 또는 일부 매도 먼저
- 슬롯 없음 → 보유 종목 수가 최대치 도달

권장 대응: `/자동사이클 점검` 또는 `/브리핑` 으로 현재 상태를 다시 확인합니다.

---

## 주문 원칙 (플랜 출력 하단에도 표시됨)

1. 매수는 장전 예약 기준입니다. 시초가 갭이 크면 추격하지 않습니다.
2. 진입 구간 위에서 출발하면 해당 날은 그 종목 건너뜁니다.
3. 1차 매도 체결 후 잔량은 2차 목표가 또는 `/보유대응` 기준으로 관리합니다.
4. 장이 약하거나 운용강도가 **보수적**이면 슬롯이 줄어드는 게 정상입니다.

## 자주 묻는 것

**Q. `/장전플랜`과 `/리포트 추천`의 차이는?**  
`/리포트 추천`은 오늘 대응할 후보 PDF를 만들어줍니다. 후보 목록과 근거 중심입니다.  
`/장전플랜`은 실제 주문에 바로 쓸 수량·가격·진입 구간까지 계산해줍니다. 텍스트로 빠르게 확인하고 바로 주문하는 용도입니다.

**Q. 매일 실행해야 하나요?**  
장 전날 밤 또는 당일 08:30~09:00 사이에 실행하는 것이 가장 정확합니다. 당일 점수 스냅샷 기준으로 계산되기 때문입니다.

**Q. 운용강도가 보수적인데 강제로 더 넣고 싶다면?**  
권장하지 않습니다. 적응형 강도는 최근 손익·승률 데이터를 반영한 결과입니다. 연속 손실 구간에서 포지션을 늘리는 것은 회복보다 리스크를 키웁니다.

**Q. 리포트 메뉴에서 어떻게 접근하나요?**  
`/리포트` → **장전플랜** 버튼, 또는 `/리포트 장전플랜` 으로 바로 실행됩니다.
