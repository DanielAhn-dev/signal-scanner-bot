# ⚠️ 신용/공매도 데이터 빠른 시작 (수동 입력)

**자동 수집 불가능** (KRX API 차단)  
→ 웹 UI에서 **매주 5분만 수동 입력**

---

## 📌 현황

**KRX API 차단으로 자동 수집 불가능**  
→ 웹 UI에서 **수동으로 매주 입력**하는 방식으로 전환

| 항목 | 상태 |
|------|------|
| 공매도 데이터 | 📝 수동 입력 |
| 신용비율 | 📝 수동 입력 |
| DB 저장 | ✅ 자동 (Supabase) |
| 차트 표시 | ✅ 자동 (최신 데이터) |

---

## 🎯 사용 방법

### 1️⃣ 웹 앱에서 버튼 클릭
"신용/공매도 데이터" → "입력" 버튼

### 2️⃣ 데이터 입력 (5분)
```
거래 날짜: 2026-05-10
공매도비율: 4.5%
신용비율: 3.2%
```

### 3️⃣ 저장
버튼 클릭 → 즉시 차트에 반영

---

## 📊 매주 언제 입력하나?

**매주 금요일 장 마감 후**  
→ Naver Finance 또는 증권사에서 공매도/신용비율 조회  
→ 웹 앱에 입력  

**또는**  

**매주 월요일 오전**  
→ 지난주 금요일 최종 데이터 입력

---

## 🔍 데이터는 어디서 조회하나?

### 공매도비율
- **Naver Finance**: stock.naver.com → 종목 검색 → "거래량" 탭
- **증권사 HTS**: 신용잔고 → 공매도 → 비율

### 신용비율
- **Naver Finance**: stock.naver.com → 종목 검색 → "신용" 탭
- **증권사 HTS**: 신용거래 → 신용비율

---

## 💡 팁

- 대량의 종목이 자동으로 저장되므로 (core + extended universe) 입력 후 즉시 모든 차트에 반영
- 정확도 100% (수동 입력)
- 자동 수집 복구 시 이전 수동 데이터와 자동 병합 가능
