# 자동매매 명령어 운영 가이드

## 📋 개요
1차부터 4차까지의 자동매매 고도화를 실제로 운영하기 위한 명령어 정리. 이 가이드를 통해 "시뮬레이션 vs 실제 실행", "장중 vs 장후 판단", "우선순위 기반 대응"을 명확히 구분할 수 있습니다.

---

## 🎯 핵심 3가지 명령어

### 1. `/자동사이클 점검`
**목적**: 🔍 시뮬레이션 (실제 주문 반영 안 함)

장점:
- 실제 반영 없이 안전하게 추가매수/신규매수 시뮬레이션
- 장중 실시간가 기반으로 수량, 매수가, 손절가 미리 확인
- 대응 우선순위 미리보기

사용 예시:
```
/자동사이클 점검
```

결과:
- 추가 매수 대상 종목과 매수가 (실시간가 기반)
- 신규 진입 대상과 수량, 예상손실, 투입비중
- 각 종목별 트레일링 규칙 (+5% 원가 기준 / -2% 손절)

---

### 2. `/자동사이클 실행`
**목적**: ✅ 실제 반영 (주문 등록)

특징:
- 실제 가상 매매에 반영됨
- 오늘 기준으로 현재 시간의 판단을 적용
- 장중(09:00-15:30): 실시간가 우선
- 장후(15:30~): 종가 기준

사용 순서:
```
1) /자동사이클 점검 으로 먼저 결과 확인
2) /자동사이클 실행 으로 실제 주문
```

주의:
- 같은 날 같은 모드 반복 금지 (중복 매수 방지)
- 실행 후 `/보유` 또는 `/거래기록` 으로 결과 확인

---

### 3. `/자동사이클 실행 진입`
**목적**: 💪 강제 신규 진입 (월요일 또는 특별 상황)

사용 시기:
- 월요일: 신규 포지션 10개 진입
- 평일: 요일 상관없이 신규 진입 강제 실행 (보유 0개 또는 기회 발생 시)

진입 내용:
- 10개 종목 신규 진입
- 각 종목: 예산 분할(1/10) + 예상손실 계산 + 트레일링 규칙 적용
- 결과: 대응가이드 카드 (기준가, 익절%, 손절%, 추가매수밴드 포함)

## 🔥 고급 사용법

### 경우 1: 보유 0개 → 신규 진입 기회
평일이지만 신규 진입을 강제하고 싶을 때:
```
/자동사이클 점검 진입
→ /자동사이클 실행 진입
```

### 경우 2: 장중 여러 번 모니터링
```
11:00 /자동사이클 점검
12:00 /보유대응
13:30 /자동사이클 점검
14:00 /보유대응
```
→ 장중 기회 발생 시에만 `/자동사이클 실행` 추가

### 경우 3: 일일 자동 사이클 비활성화
- 보유 종목이 너무 많거나
- 리스크가 높을 때
→ 매일 `/자동사이클 점검` 만 실행, 필요 시만 수동 실행

## 📊 명령어 결과에서 보이는 4차 고도화 효과

### 1차 개선: 실시간가 기반 매수가
**문제**: OCI홀딩스가 20일에 30만원 미도달인데도 30만원으로 매수됨
**해결**: 장중(09:00-15:30)은 Naver 실시간가, 장후는 종가로 자동 전환

결과:
```
[실행 매수안]
삼성전자 (기준: 50,000원 ← 실시간가)
```

---

### 2차 개선: 명령어 명확화
**문제**: "실행"과 "진입"의 차이 불명확
**해결**: 명령어 분리 + 결과에 설명 추가

결과:
```
/자동사이클 실행: 오늘 기준, 일일 추가매수
/자동사이클 실행 진입: 신규 포지션 진입
/자동사이클 점검: 시뮬레이션 (실제 반영 X)
```

---

### 3차 개선: 트레일링 규칙 + 포맷 통일
**내용**: 각 대응 가이드에 트레일링 규칙 명시

결과:
```
[대응 가이드]
삼성전자
- 기준: 50,000원
- 익절: 52,500원 (+5%)
- 손절: 47,500원 (-5%)
- 추가: 48,000~50,500원 (-4% ~ +1%)
- 트레일링: +5% 원가 기준 / -2% 손절
```

---

### 4차 개선: 우선순위 정렬 + 리스크 메트릭
**내용**: 손절 필요 > 손실액 큰 종목 우선 정렬

결과:
```
대응 카드 정렬 기준:
1순위: 손절 급급한 종목 (손절 거리 가까움)
2순위: 예상손실 큰 종목
3순위: 익절 기회 종목
4순위: 보유 관찰 종목

각 카드에 표시:
- 수량: 285주
- 예상손실: 8,500원
- 투입비중: 4.2%
```

## 🔍 결과 해석 가이드

### [대응 가이드] 카드 읽기
```
[대응 가이드]
삼성전자 (기준: 50,000원)
- 익절 목표: 52,500원 (+5.0%)
- 손절 목표: 47,500원 (-5.0%)
- 추가매수 밴드: 48,000~50,500원 (-4% ~ +1%)
- 트레일링 규칙: +5% 원가 기준 / -2% 손절

🔴 리스크 분석
- 수량: 285주
- 예상손실: 8,500원 (손절 시)
- 투입비중: 4.2%
```

읽는 방법:
1. **기준가**: 현재 기준으로 대응 판정 시점의 주가
2. **익절 목표**: +5% 시 매도 (원가 기준)
3. **손절 목표**: -5% 시 손절 (원가 기준)
4. **추가매수 밴드**: -4% ~ +1% 범위에서 추가매수 가능
5. **트레일링**: +5%에 도달하면 손절가를 원가 이상으로 올림 (-2% 유지)
6. **수량**: 이번 포지션의 주식 수
7. **예상손실**: 손절 시 최대 손실액
8. **투입비중**: 전체 포트폴리오에서 이 종목의 비중

---

### [보유대응] 결과 읽기
```
최우선 점검: 삼성전자 -5.8% (예상손실 45,000원)
→ 손절 임박. 즉시 확인 필요

우선 익절 점검: SK하이닉스 +4.9% (예상손실 12,000원)
→ 익절 가능 범위. 추가 상승 시 매도 판단

관찰 유지: LG전자 +1.2% (예상손실 8,000원)
→ 손절/익절 거리 중간. 계획 유지
```

정렬 기준:
1. **최우선 점검** (손절 점수: 3000점 + 예상손실액)
2. **우선 익절 점검** (익절 점수: 2000점 + 예상손실액)
3. **관찰 유지** (보유 점수: 1000점 + 예상손실액)

## 💬 FAQ

**Q: 점검과 실행 중 어느 것을 자주 써야 할까요?**
A: 보수적으로 운영하면 점검 위주, 활발하면 매일 실행. 처음에는 점검으로 시작한 후 결과를 확인하고 실행하는 것을 권장합니다.

**Q: 실행 진입은 월요일에만 써야 하나요?**
A: 월요일이 기본이지만, 보유 0개가 되었을 때도 사용할 수 있습니다. 평일에 강제 신규 진입이 필요한 경우입니다.

**Q: 같은 날 점검과 실행을 여러 번 해도 되나요?**
A: 점검은 무제한 가능합니다. 실행은 같은 모드 반복을 권하지 않습니다 (중복 매수 방지).

**Q: 장중과 장후에 차이가 있나요?**
A: 맞습니다. 장중(09:00-15:30)은 실시간가, 장후는 종가 기준입니다. 더 정확한 판단을 원하면 장중에 /자동사이클 실행을 추천합니다.

**Q: 예상손실과 투입비중이 뭐예요?**
A: 예상손실은 손절 시 잃는 돈, 투입비중은 이 종목에 투입한 전체 자본 비중입니다. 이 두 가지로 리스크를 파악할 수 있습니다.

## 📞 문제 해결

**문제**: 실행했는데 주문이 반영 안 됨
→ 보유 상한(보통 10개)에 도달했거나, 자본 부족 가능성. `/보유` 확인.

**문제**: 점검과 실행 결과가 다름
→ 실행 시간이 다르면 실시간가가 바뀌므로 결과 달라질 수 있습니다. 점검은 참고만 하세요.

**문제**: 트레일링 규칙이 적용 안 됨
→ 아직 +5% 수익에 도달하지 않았을 수 있습니다. 기준가 대비 +5% 도달 후 적용됩니다.
