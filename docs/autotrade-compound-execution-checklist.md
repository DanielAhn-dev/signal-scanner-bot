# 자동매매 복리형 고도화 실행 체크리스트

목적: 단일 익절(예: 5%/10%) 고정 전략에서 벗어나, 종목별 목표수익률/보유기간을 동적으로 운영하고 강한 리더 종목은 장기 트랙으로 분기한다.

업데이트 규칙
- 작업 시작 시 체크박스 상태를 확인한다.
- 작업 종료 시 완료 항목 체크, 미완료 이유를 메모한다.
- 다음 대화 시작 시 마지막 섹션의 다음 액션부터 이어간다.
