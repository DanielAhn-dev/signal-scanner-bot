## 리포트 명령

- /리포트: 가능한 리포트 종류와 버튼 메뉴 표시
- /리포트 주간: 시장 + 포트폴리오 종합 PDF
- /리포트 포트폴리오: 보유 종목/거래 중심 PDF
- /리포트 거시: 거시 지표 PDF
- /리포트 수급: 자금 흐름 PDF
- /리포트 섹터: 섹터 강도 PDF
